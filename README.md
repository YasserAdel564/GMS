# GMS

GMs is Mediacl App For Watch Different Courses related with Medical Field 

## Features
- [x] Kotlin
- [x] Kotlin Coroutines
- [x] MVVM Architecture
- [x] Ksoap2
- [x] Room db
- [x] Repository Design Pattern
- [x] DI(Hilt)
- [x] View Binding
- [x] Navigation Components
- [x] Google Material Design
